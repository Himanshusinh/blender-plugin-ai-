# AI Render Addon for Blender

This directory contains a Blender addon `AiRender07.py`.

## Running the Addon (Developer Mode)

To run the addon directly without installing it, use the provided script:

```bash
./launch_addon.sh
```

This will launch Blender with the addon loaded. You can find the panel in the 3D Viewport > Sidebar (N) > **AI Render**.

## Installing Permanently

For a permanent installation:

1. Open Blender.
2. Go to **Edit > Preferences > Add-ons**.
3. Click **Install...** at the top right.
4. Select the `AiRender07.py` file.
5. In the list, enable the checkbox for "Render: AI Render â€“ Wildchild Studios (Final)".
6. Save Preferences (if not auto-saved).

This ensures the addon is loaded every time you start Blender.
